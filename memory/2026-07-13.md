## What changed
- **Theme overhaul (`assets/style.css`, `index.html`)** — replaced the playful,
  colorful design (lavender bg, animated rainbow-gradient title, drifting blobs,
  rounded per-topic-colored cards) with a Swiss/editorial directory: white page,
  one heavy black `digster` wordmark, a single red accent (`--accent`), a numbered
  (`01…NN`) three-column catalog grid separated by hairline rules, squared `#tag`
  chips, mono-caps labels ("Project Index / NN", "Visit site →", "Source ↗"), and
  a compact top-right avatar + moon toggle + search. Dark mode retained (inverted
  editorial palette). Removed the blob markup and all decorative animations.
- **`assets/app.js`** — rewrote `createCard()` to render numbered entries; `render()`
  now fills `#index-count` from `sites.length`; `applyFilters()` selector `.card`
  → `.entry`; deleted the per-topic color machinery (`TAG_HUES`, `hashString`,
  `hueFor`, inline `--h`). Search, tag filtering, theme toggle and error/empty
  states are unchanged.
- **Content** — footer → "Built with ❤️"; masthead subtitle → "Cogito, ergo sum"
  (kept "A directory of small web experiments" as the `<title>`/meta description).
- **New project (`data/sites.json`)** — added **3D Physics Engine**
  (`3d-physics`, https://digster.github.io/3d-physics/), an interactive C++20/SDL3
  tutorial that builds a physics engine from scratch. It was the only repo on the
  account created after the directory was built with GitHub Pages enabled and not
  already listed (verified `has_pages: true`). `sdl3-2d` was checked and excluded
  (`has_pages: false`; source-only, already covered by "SDL3 Recipes"). Directory
  is now 10 projects.
